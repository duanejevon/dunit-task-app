## Decision & Rationale
**Chosen stack: Electron + React + TypeScript + `better-sqlite3`**

- Electron: mature ecosystem, full Node.js access for storage, well-tested drag-and-drop libraries (`dnd-kit`), straightforward native notifications for reminders. Tradeoff is bundle size (~100MB+), which doesn't matter for a single-user personal app.
- Tauri: smaller bundle and lower memory, but relies on inconsistent system webviews across OSes, a Rust/JS bridge to learn, and a thinner plugin ecosystem for local notifications/SQLite.
- For a personal, offline-only kanban tracker, dev speed and ecosystem maturity outweigh bundle size — Electron wins.
